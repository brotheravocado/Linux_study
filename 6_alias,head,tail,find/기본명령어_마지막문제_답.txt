1. /etc/passwd 의 위에서 5번째 줄까지 출력합니다 
head -5 /etc/passwd | cat -n
cat -n /etc/passwd | head -5

2. inittab 파일 검색 후 별도로 아래서부터 12번째 줄까지 출력합니다 
tail -12 /etc/inittab | cat -n
cat -n /etc/inittab | tail -12

3. /etc/httpd/conf/httpd.conf 문서를 20줄씩 끊어서 출력합니다
more -20 /etc/httpd/conf/httpd.conf | cat -n	<== 이렇게 하면 1009줄이 행번호 붙여서 전체 출력 된다  X
cat -n /etc/httpd/conf/httpd.conf | more -20	<== 정답

4. /etc/httpd/conf/httpd.conf 를 왼쪽에 번호표가 붙은 상태로 10줄씩 끊어서 출력합니다
cat -n /etc/httpd/conf/httpd.conf | more -10

5. /etc/ 디렉토리의 목록을 2줄씩 끊어서 출력합니다
find /etc -type d | more -2

6. /etc/ 디렉토리의 목록을 왼쪽에 번호표가 붙은 상태로 출력합니다
find /etc -type d | cat -n

** 7번부터는 /cattest 디렉토리 안에서 작업합니다 **

7. /cattest 디렉토리를 생성한뒤 pwd 를 입력했을때 /cattest 가 나올 수 있도록 합니다 
mkdir /cattest
pwd

8. 가로로 1 2 3 내용을 가지는 파일 a와 세로로 1 2 내용을 가지고 b 파일을 만듭니다
[root@kgitbank cattest]# cat > a
1 2 3
^C
[root@kgitbank cattest]# cat > b
1
2
^C

9. 파일 a 와 b 내용을 병합한 새로운 파일 c를 생성합니다 
cat a b > c

10. kgitbank 내용을 가지는 파일 d를 생성한 뒤 파일 c의 내용을 d파일에 추가합니다
[root@kgitbank cattest]# cat > d
kgitbank
^C
[root@kgitbank cattest]# cat c >> d 

11. /etc 왼쪽에 번호표를 가지면서 디렉토리의 목록의 내용을 가지는 파일 e를 생성합니다
find /etc -type d | cat -n > e

12. grub.conf login.defs inittab 파일을 한꺼번에 검색하여 /cattest 로 복사하고
	grub.conf,login.defs,inittab,a,b,c,d,e 파일들의 위에서 각 파일마다 2줄씩만 내용이 출력되는
	파일 f를 /home/test/c/d/cattest/linux 디렉토리에 생성합니다
mkdir -p /home/test/c/d/cattest/linux && head -2 * | cat -n > /home/test/c/d/cattest/linux/f
