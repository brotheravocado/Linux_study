head = 파일의 내용중 처음부터 아래로 10줄을 출력 (10줄이 기본값)
head -n /etc/passwd  = /etc/passwd 의 내용을 위에서부터 n줄만큼 보여달라
--------------------------------------------------------------------------
tail = 파일의 내용중 마지막부터 위로 10줄을 출력 (10줄이 기본값)
tail -n /etc/passwd  = /etc/passwd 의 내용을 마지막부터 위로 n줄만큼 보여달라
------------------------------------------------------------------------------
more =less  = 내용이 많은 파일을 ★화면단위★로 끊어서 출력 = dir/p 
 엔터키 = 한줄씩 출력
 스페이스키 = 한페이지씩 출력
ex) ls -l /etc/ |more

rdate = 타임서버에서 시간 정보를 얻어 시스템의 시간을 변경함
rdate -p  time.bora.net  = 타임서버의 현재시간 확인
rdate -s  time.bora.net = 타임서버와 시간 동기화 = sync

zero.bora.net

gps.bora.net
ntp1.cs.pusan.ac.kr
ntp.ewha.net

--------------------------------------------------------------------------------
파일유형 (Type)
- 	파일
d	디렉토리
l	심볼릭 링크 파일
b	블록형 특수 장치파일(주로 디스크, /dev/sda 등)
c	문자형 특수 장치파일(주로 입출력에 사용, /dev/console 등)
p	파이프 파일임을 의미
s	소켓 파일임을 의미
.file	file앞에 붙는 . 은 숨김을 의미
.	현재 경로
--------------------------------------------------------------------
입/출력에 관한 정보
1	정상출력
2	에러출력

입출력에 사용하는 기호
1>	쓰기
1<	읽기
<<	추가
2>	에러출력
1>&2	정상출력을 에러출력
2>&1	에러출력을 정상출력
--------------------------------------------------------------------
예제)어느 디렉토리든 일단 가셔서 touch 1 2 3 만 하시고 아래의 내용을 테스트해 보세요

cat 1 2 3 4 1> 5		정상내용만 입력
cat 1 2 3 4 2> 5		오류내용만 입력, 정상은 출력
cat 1 2 3 4 > 5 1>&2 		=> 성공 오류내용 전부 출력 (오류로 빠짐)
cat 1 2 3 4 > 5 2>&1 		=> 성공 오류내용 전부 입력
